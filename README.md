
**Blog**
This project presents a blog for sharing your posts. On technical side it uses Django 1.11, Python 3.6. 
**Features:**

 1. Drafts
 2. Posting images
 3. Searching through posts (includes search for author, context and title)
 4. Pagination
 5. Disqus comments
 6. VK and FB sharing
